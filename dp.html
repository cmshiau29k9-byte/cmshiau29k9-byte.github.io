🔶 DP 認證簡介

DP 為 Data Professional Certification 之縮寫，由全球數據協會 (GDA) 主辦，涵蓋資料分析、資料科學及相關實務技能。分為基礎級 (DP-Foundation) 與專業級 (DP-Expert) ，適合分析師、科學家及轉型中的數據從業人員。

📋 認證領域 (專業級方向)

  • 數據分析與視覺化 (Tableau, Power BI, 統計檢定, 資料敘事)
  • 機器學習實務 (監督/非監督學習、模型評估、特徵工程)
  • 資料處理與SQL (進階查詢、效能調校、資料清洗)
  • 數據倫理與治理 (隱私、偏見、可解釋性)
  • 大數據基礎 (Spark SQL, 雲端分析服務)

📌 認證方式

DP 認證採模組化考試：基礎級為 90 分鐘選擇題；專業級需通過兩科專業選考 (例如「視覺化+機器學習」) 並提交實作作品集 (GitHub 或分析報告)。

  • DP-100：數據分析核心 (SQL, 統計, 可視化)
  • DP-200：資料科學與機器學習實戰
  • 選考科目 (任選一)：雲端分析 / 資料工程基礎 / 商業智慧

💼 職場價值

DP 認證受中小型企業及新創公司青睞，代表持有者具備端到端的數據解題能力。持有 DP 專業級者常擔任數據分析主管、資料科學家或顧問角色。在履歷中加入 DP 認證可增加跨領域機會。
